## [1.0.0] - 2025-02-01

### Added

- Initial release
- `RaynaUILineIcons` class with outline/stroke style icons
- `RaynaUIFillIcons` class with solid/filled style icons
- Cross-platform support (Android, iOS, Linux, macOS, Windows, Web)
- Tree shaking support for optimized builds
